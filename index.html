版本：v

使用說明：

  • 📊 總覽 — 每月花費統計，◀ ▶ 切換月份，分類圓餅圖 + 明細表
  • ➕ 記錄 — 點標題可自訂 App 名稱 · 🌙 切換深色主題
  •     📍 GPS 自動定位目前位置，或 🔍 搜尋地標名稱
  •     📷 拍照或選相簿，最多 3 張，選取後自動壓縮，✕ 可移除
  •     💰 選擇幣別自動換算台幣，🔄 可手動更新即時匯率
  •     ⭐ 1-5 星評價 · 📝 隨筆備註（500 字）
  •     📌 分類可選系統分類或「自訂...」輸入新分類
  • 📋 列表 — 按日期分組，每日小計一目了然，上方分類篩選
  • 📤 匯出 — Markdown 下載 · JSON 備份（含照片）/還原 · CSV（Excel）
  • 🖼️ 照片 — 點擊照片可放大檢視（lightbox），再點關閉

📁 資料安全：所有資料存於本機（IndexedDB），不上傳任何伺服器

💱 匯率來源：open.er-api.com（即時抓取，7 天快取，離線可用最近記錄）

🗺️ 地圖資料：OpenStreetMap Nominatim（免費地理編碼服務）
